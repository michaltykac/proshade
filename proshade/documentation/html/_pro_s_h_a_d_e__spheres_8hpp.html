Detailed Description

This header file contains the declarations for the ProSHADE_sphere class.

The ProSHADE_sphere class and the ProSHADE_internal_spheres namespace are both declated in this header file. The functionalities that these two objects provide to ProSHADE are mapping a 3D map onto a surface of a sphere with a given radius, object for holding such a mapping and onto which the spherical harmonics decomposition could be done as well as support function for data access and processing.
