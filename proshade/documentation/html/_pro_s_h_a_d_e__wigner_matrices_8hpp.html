Detailed Description

This header declares the functions required to compute the Wigner D matrices.

The functions declared in this header file in the ProSHADE_internal_wigner namespace are required to compute the Wigner D matrices, which in turn allow the computation of the inverse SOFT transform as well as rotation of internal map representations in the spherical harmonics coefficient space.
